# Resources

This section lists recommended resources for each module of the roadmap.

## Math & Statistics
- Khan Academy: Linear Algebra, Calculus
- 3Blue1Brown: “Essence of Linear Algebra” series
- OpenIntro Statistics (free textbook)

## Python & Data Analysis
- *Python for Data Analysis* by Wes McKinney
- *Automate the Boring Stuff with Python* by Al Sweigart
- Official pandas and NumPy documentation

## Machine & Deep Learning
- *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow* by Aurélien Géron
- Andrew Ng’s Deep Learning Specialization (Coursera)

## GenAI & RAG
- OpenAI Cookbook examples
- LangChain documentation

Feel free to suggest additional high-quality resources via pull requests.
